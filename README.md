# HerbsZen Premium E-Commerce Website

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/iosvarun/herbsZebWebsite)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3005](http://localhost:3005) with your browser to see the result.

## Deploying to Render (1-Click)

1. Click the **"Deploy to Render"** button above.
2. Sign in to your Render Account.
3. Paste your Google App Password for `SMTP_PASS` in the prompt, and click **"Apply"**.
4. The website will build and go live automatically!
